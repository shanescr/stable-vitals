# stable-vitals
Stable Vitals is a smart equine hydration monitoring system designed to track hourly water intake per horse via automatic waterers and deliver actionable alerts when intake is abnormally low or high.

## What STAN Does
- Measures water flow via Hall-effect sensor
- Aggregates hourly and daily intake
- Sends alerts for abnormal hydration patterns
- Operates on Wi-Fi with local buffering
- Battery-powered (AA lithium) with optional USB-C power

## What STAN Does NOT Do
- Medical diagnosis
- Multi-horse shared water tracking
- Native mobile apps (v1)
- Real-time streaming dashboards
